Descripción

Sistema de Detección de Similitud entre Sílabos de Instituciones de Educación Superior (IES) del Ecuador

  • Creación del repositorio semántico de sílabos de IES del Ecuador
  • El sistema extre sílabos de base de datos tradicionales de las instituciones, transforma estos datos a formato RDF, los almacena en un repositorio de Tripletas (Reositorio Semántico).
  • Sistema de detección de similitudes
  • A partir de los datos RDF, se ejecutan técnicas de minería de datos para obtener un valor de similitud entre pares de sílabos
  • Un sílabo está compuesto de varias secciones, entre ellas el "Contenido Académico". El Contenido académico de un sílabo esta compuesto por capítulos y subcapítulos. Es esta sección la que tiene mayor peso al momento del análisis de similitud.

Este sistema está basado en la aplicación de tecnologías semánticas y minería de textos
